# AGENTS.md

## Commit Messages

Use conventional commit subjects for every commit in this repository.

Required format:

`type(scope): message`

Rules:

- Use lowercase `type` and `scope`.
- Keep the subject concise and imperative.
- When amending, rebasing, or squashing commits, preserve this format.

Examples:

- `feat(runtime): add Kotlin runtime and plugin tooling`
- `fix(cli): handle missing config path`
- `docs(readme): clarify quick start`
